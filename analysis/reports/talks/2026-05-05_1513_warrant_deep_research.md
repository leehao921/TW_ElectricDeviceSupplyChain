### 3.3 部位 sizing — Modified Kelly

**Kelly fraction (使用 089065 為例):**
- p(獲利) = 40%, b = 平均獲利/平均虧損 = 67% / 44% = 1.52
- Kelly% = (p·b - q) / b = (0.40 × 1.52 - 0.60) / 1.52 = **0.5%**

→ Kelly 算出來是 0.5% NAV，但因為 binary event 不確定性高，**用 1/4 Kelly = 0.125% NAV**？太保守。

**實務建議：用「最大可承受全損部位」反推**
- 若 5/12 利空全損，可承受最大損失 = NAV × 2.5%（已含心理痛閾）
- 若 NAV = 200 萬 → 上限 5 萬

**情境:** 5/12 +8%、依規則砍 50%、5/13 早盤 -5%

2. 若現股回到 5/12 收盤價 ±1% 內 → 持有剩餘部位 24 小時
3. 若現股續跌 -2% 以上（vs 5/12 收盤）→ **立即出場**（典型「sell the news」）
4. 若已加碼（不該發生，但若發生）→ **加碼部位 break-even 即出**，不要凹回成本

**EV = NT$ -3,000（虧損 5%）**
**95% VaR = NT$ -28,000（虧損 47%）**
**最大可能虧損 = NT$ -36,000（全損 60%，假設停損失效）**

### 決策邏輯

我推薦這個方案不是因為期望值正，而是因為：
1. **尾部報酬 +99% 在合理機率（15%）內可達** — 真正的「lottery ticket」
2. **這兩檔的 IV 是市場最便宜（50-53%）**，即使賠也賠得比別人少
3. **平價附近 + 中天期 (80-98 天)** — 不被 theta 急殺
4. **部位上限 2.5% NAV** — 全損仍可承擔，不影響核心組合

如果你 **不能接受 60% 全損的可能**，**請改買現股 1513 @ 150-153** — 那才是這份分析最終背書的「核心配置」（見 deep_dive 報告 §11）。權證僅為「事件衍生 alpha」，不是核心。
