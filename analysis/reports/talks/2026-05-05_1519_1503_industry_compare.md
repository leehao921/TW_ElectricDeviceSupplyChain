## 1. 1519 華城 — 「全球變壓器供應鏈龍頭」

### 1.1 業務組合（從 Pilot Report 拆）

| 業務 | 估算佔比 | 毛利率 | 戰略地位 |
|---|---:|---:|---|
| **超高壓變壓器 500kV** | **70-75%** | **42-45%** ⭐ | 主力 + 美國市場 |
| 配電盤 / EPC | 15-20% | 30-35% | 國內配套 |
| 維修 + 服務 | 5-10% | 50%+ | recurring |
| 海外特殊案 (星際之門) | 5%+ | 高 | 一次性大單 |

### 1.2 上游供應鏈（高度依賴日韓）

| 原料 | 主要供應商 | 議價力 |
|---|---|---|
| **矽鋼片** | **日本製鐵 / POSCO** ⚠️ | **賣方極強** |
| **銅材** | 華新麗華 (台) | 平衡 |
| **絕緣油 / 樹脂** | 國際大廠 | 賣方強 |
| 機殼 / 鑄鐵 | 國內中小廠 | 中興電有議價力 |

> ⚠️ **致命弱點:** 矽鋼片 = 變壓器靈魂材料，日本製鐵 + POSCO 兩家寡占。日韓若跟進美國加關稅 → 直接打擊毛利率。

### 1.3 下游客戶（黃金分散）

| 客戶 | 估算佔比 | 戰略意義 |
|---|---:|---|
| **🇺🇸 美國 17 家電力公司** | **50%+** ⭐⭐⭐ | AEP / PG&E / Duke / Southern |
| **🇹🇼 台電** | 20-25% | 強韌電網 |
| **🏭 半導體 (TSMC)** | 10-15% | 美廠 + 台廠擴張 |
| **🤖 AI 資料中心 (星際之門)** | **5-10% 增長中** | OpenAI / SoftBank / Oracle 主導 |
| **🌬️ 離岸風電 (沃旭)** | 3-5% | 升壓站 |

→ **單一客戶集中度 < 5%** 是三雄裡最低，**這是 P/E 30x 高估值的根本原因**。

### 1.4 5 道護城河

| 障礙 | 強度 |
|---|---|
| **美國 17 家電力公司 5+ 年認證** | ⭐⭐⭐⭐⭐ |
| **500kV 製造能力**（台廠唯一） | ⭐⭐⭐⭐⭐ |
| 與星際之門/AI hyperscaler 合作 | ⭐⭐⭐⭐ |
| 與沃旭等離岸風電綁定 | ⭐⭐⭐ |
| 25 億擴廠投資 (2027 Q2 上線) | ⭐⭐⭐ |

### 1.5 風險
- 🔴 **矽鋼片日韓壟斷** + 美關稅升級
- 🔴 **TWD 對 USD 貶值** = 美國訂單匯損（2024 賺、2025-2026 受壓）

## 2. 1503 士電 — 「多元產品線 + 三菱技術合作」

### 2.1 業務組合（最分散）

| 業務 | 估算佔比 | 毛利率 |
|---|---:|---:|
| **重電（變壓器/配電盤）** | 35-40% | 25-30% |
| **車輛零件**（電動巴士/車用 motor） | 20-25% | 20-25% |
| **馬達 / 自動化** | 15-20% | 18-22% |
| **數位家電** | 10-15% | 15-20% |
| **半導體製程設備** | 10-15% | 25-30% |

### 2.2 上游 + 三菱合作（雙刃）

| 來源 | 內容 | 性質 |
|---|---|---|
| **🇯🇵 三菱電機 Mitsubishi Electric** | 技術授權、特定零組件 | **戰略夥伴 + 限制** |
| 銅 / 鋁 / 矽鋼片 | 國際大廠 | 一般採購 |

> ⭐ **三菱技術合作的雙刃性質:**
> - ✅ 帶來日本級技術品質
> - ❌ 海外接單 (尤其美國) 受三菱地理區域分配限制
> - ❌ 部分高利潤產品需付授權費

### 2.3 下游客戶（最分散）

| 客戶 | 佔比 | 屬性 |
|---|---:|---|
| **🇹🇼 台電** | 20-25% | 強韌電網 |
| **🏭 半導體 (TSMC/UMC/Micron)** | 25-30% | 廠房擴建 |
| **🚗 車用 (三陽/光陽/中華車)** | 15-20% | 機車馬達/車用 |
| **🇺🇸 北美電力 + Mitsubishi 系統** | 10-15% | 透過三菱接單 |
| **🏗️ 建築 + 工業** | 15-20% | 一般電裝 |

→ 最分散，但**因為什麼都做、什麼都不專精**，毛利率僅 20.9% 是三雄最低。

### 2.4 護城河（中等）

### 3.2 三家的客戶結構對比（重要）

| 客戶類型 | 1519 華城 | 1513 中興電 | 1503 士電 |
|---|---:|---:|---:|

## 6. 投資邏輯結論 — 三家如何取捨
