# MLCC 008004 — TW 真實受惠拆解 + Pair Trade 思路

> **008004 = imperial 008004 = metric 0201 = 0.25 × 0.125 mm**,目前 [[MLCC]] 最極致量產規格,全球僅 Big-4 ([[Murata]] / [[Samsung Electro-Mechanics]] / [[Taiyo Yuden]] / [[TDK]]) 能做。**TW 中游廠商 零直接 exposure**,真正受惠的是兩家代理通路 (8043 / 3090) 加上一個材料 catch-up 故事 (6173 / 4760)。
> 完整族群供應鏈圖見 **[[../../themes/MLCC.md]]**。

## Catalyst (建立本概念頁的觸發點)

**2026 年 5 月 [[國巨]] (2327) 與 [[華新科]] (2492) 連續漲停 +175% / +194%**,3-year valuation percentile 衝到 99.9%。市場 narrative 三件事疊加:(a) MLCC commodity 漲價週期 (b) AI 伺服器 [[MLCC]] 高用量 (c) 「next-gen MLCC = 008004」**錯誤掛勾**。本概念頁專責拆解後者 — **008004 與 2327/2492 漲停 narrative 沒有正向關聯**,但市場/媒體大量混用。

## 關鍵 finding — 用戶假設驗證

| 命題 | 驗證 |
|---|---|
| 「2327/2492 = 008004 受惠」 | **錯誤**。公開研究 ([Vocus](https://vocus.cc/article/6a0b27bafd897800010b06e7)) 直言「008004 由 [[Murata]] 獨占,台廠 1-2 世代落後」。2327 最小 01005 imperial,2492 最小 0201 imperial,無 008004 量產或公開 R&D 證據 |
| 「漲價 narrative = 008004 切入」 | **錯誤**。Murata 4/1 +15-35% / Taiyo Yuden 5月 +6-13% 為 commodity 漲價,主因白銀價格 + AI 伺服器把高階產能擠出消費級,**不代表台廠進入 008004 技術節點** |
| 「AI server BOM 高用量 = 008004 高用量」 | **部分錯**。[[GB300]] 一台 ~45 萬顆 MLCC,但 008004 估只占顆數 **5.6%**,占 MLCC 單櫃 ASP **45%** (NT$112.5K / NT$248.75K)。其餘 94% 是 0201/0402/01005 大宗 |
| 「LONG 8043 / SHORT 2492 pair trade」 | **方向對、對立面選錯**。兩檔同吃 MLCC 漲價週期 β,沒真正對沖。更乾淨:**LONG 8043 / SHORT 2327** (2327 處置股 + PER 65×) 或 **LONG basket {8043 + 3090 + 6173}** |

## Pair Trade 思路

### Option A: LONG basket {8043 + 3090 + 6173} (推薦)

- 純 long,等權配置 33%/33%/33% (or 加 [[4760]] 改 25%×4)
- 預期持有 6 個月
- **退場訊號**: [[Murata]] 006003 量產時間正式公布 (估 2026 末–2027 H1)

### Option B: LONG 8043 / SHORT 2327

- 比 LONG 8043 / SHORT 2492 更乾淨 (2327 = 處置股 + PER 65× + commodity beta 更純)
- catalyst: 2327 處置股鬆綁 + 法說會 commodity 漲價放緩

### 為何 LONG 8043 / SHORT 2492 不對焦

1. 兩檔同吃 MLCC 漲價週期 β,沒對沖
2. 2492 +173% 是 commodity + 估值修復,跟 008004 narrative 沒正反關係
3. 2492 NI −23% YoY 但漲不停 = 估值題材股,short 易被軋

## Open questions

1. **[[Murata]] 006003 正式量產時程?** 公開資料 2024/09 CEATEC 發表,但官方未公告大量產日期 — 追蹤 2026 H2 Murata IR / CEATEC 展會
2. **8043 拆出 008004 SKU 占代理 revenue 比?** 法說只揭 AI server 40%,未細分 008004 vs 0201/0402
3. **4760 勤凱 銅漿粒徑等級 D50/D90?** 公開資料未揭露,無法判斷是否符合 008004 規格 (端電極 Cu paste 通常需 D50 < 500 nm)
4. **6173 信昌電 100 nm 目標是 X7R 還是 NP0?** 兩者製程不同,NP0 更難
5. **GB300 / VR200 008004 BOM 比例**: 5-10% 為估算,需 ODM ([[廣達]] / [[緯穎]]) 拆機或 NVIDIA 簡報驗證
6. **2327 / 2492 是否有非公開 008004 R&D?** ECTC/CIPS 論文或 Apple/Samsung PoP 拆解報告值得追蹤
