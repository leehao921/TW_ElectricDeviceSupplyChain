# Unit 8: 6173 信昌電 + 4760 勤凱 上游材料深度驗證

## TL;DR

本輪深挖五大 open question，**沒有任何一家公開資料可獨立確認 6173 / 4760 已成為 [[Murata]] / [[Taiyo Yuden]] / [[Samsung Electro-Mechanics]] 的 008004 line in-spec 供應商**。但有以下高信度收斂：

1. **6173 的 100nm 目標明確是 [[NP0]]（即 [[C0G]]）等級高純度粉**，而非 [[X7R]]——這是技術新聞 + 法說會的雙重佐證。TechNews 標題即點明「掌握高端 [[NP0]] 陶瓷粉末」，法說會直接將「200nm 以下超細粒徑」與「[[NP0]] 對標日系水準」綁定。**Prior batch 把粒徑路線當作通泛規格升級是不完整的——目標客群是高壓 [[NP0]] 應用，而非微型化 [[X7R]] 消費市場。**
2. **6173 法說會明確自承「避開日系廠商擅長的 0201 等微型化手機應用」**，主攻 1206 以上大尺寸高壓——這對 008004 假設形成**直接矛盾**：6173 不打算用 100nm 粉切 [[Murata]] 008004 line，而是用 100nm 粉做更高壓的 1206/1812/2220 [[NP0]]。**這修正了 prior batch 的核心假設。**
3. **4760 的「日系認證」是法說自承的「送件日系 MLCC 龍頭、通過初步認證階段」**，並未明確點名 [[Murata]] 或 [[Taiyo Yuden]]，業界口徑高度暗示是 [[Murata]]（「日系龍頭」+「銅膏」+「初步階段」）。**規格未公開——絕無 008004 的直接證據**，且 4760 強項是配方客製化，並非粒徑競爭。
4. **4760 月增 20% 的銅漿訂單對應 prior batch 解讀有誤**——實為「**2025 銅膏 / 銀膏全年雙雙約成長 20%**」（同步成長率），非單月 MoM。客戶結構 ≈ 台灣 70%+（[[國巨]] 銀膏佔 50%、[[奇力新]] 銀膏佔 60-70%、[[華新科]] 約 10%）、中國次之、韓國第三、日本仍在「耕耘中」。
5. **Ni paste 台灣完全沒有公開上市玩家可替代 [[Shoei Chemical]]**。[[科揚]] 僅代理日本 [[MMS]]（Mitsui Mining & Smelting）的 nano Ni slurry，沒有自研。**這是 TW 切 008004 中下游 BME 製程最大結構性卡點**——比鈦酸鋇粉更嚴重，因 BME 內電極佔成本 35-50%，且 Ni 粉 / 鎳膏沒有 toll refining 之外的替代路徑。

**結論：6173 + 4760 兩家確實是 TW 最接近上游材料突破的標的，但是「攻 008004 line」的敘事在公開資料上是 untestable。更可信的 thesis 是「攻大尺寸 / 高壓 / [[NP0]] / [[Mega Cap]] / TGV / AI PSU」——這是與日系 [[Murata]] / [[Taiyo Yuden]] 在 008004 上正面對撞的方向之外、互補性更強的賽道。**

## Q1: 6173 100nm 目標是 [[X7R]] 還是 [[NP0]]？

### 結論：高信度為 [[NP0]] / [[C0G]]（不是 [[X7R]]）

### 證據

**TechNews 標題即定錨：**
> 「信昌電六甲新廠估明年第三季投產！掌握高端 **[[NP0]]** 陶瓷粉末攻 AI 訂單」<br>
> 來源：https://finance.technews.tw/2026/05/15/high-end-np0

**法說會（2026-05-21）內文：**
- 「NP0 系列材料效能優異，應用於 PSU、BBU 等關鍵電源迴路」
- 「[[NP0]] 約佔 MLCC 營收 10%，單價為同尺寸普通品 3 倍以上」
- 「[[NP0]] 產品**貼近日系水準**，仍有進步空間」憑粉末自製維持競爭力
- 來源：https://finance.biggo.com.tw/news/TW_6173.TW_2026-05-21

**粒徑路線：**
- 法說會口徑：「技術目標鎖定 **200nm 微細化粉末**」
- 媒體口徑（TechNews / 壹蘋）：「粉體粒徑已從過去 600~800 奈米，**逐步朝 100 奈米等級發展**」「挑戰日系大廠水準的 100 奈米極細粉末」

> 公司公開法說口徑是 200nm，產業媒體把終局目標寫成 100nm。**兩者並不衝突——200nm 是 2027 Q3 六甲投產的階段性規格，100nm 是 follow-on 路線**。

**為何是 [[NP0]] 而非 [[X7R]]：**
1. [[NP0]] 為 Class-I 介電質（K=10~100），對 [[鈦酸鋇]] 純度要求遠高於 [[X7R]]（Class-II, K=1000~5000）——TW 自製 100nm 粉若做 [[X7R]] 是技術降階，做 [[NP0]] 才是真正對標 [[Sakai 化學]] / [[Nippon Chemical]] 寡占帶。
2. [[NP0]] 對應「LLC 共振迴路高頻 / 高穩定」應用——即 6173 法說 7-9x ASP 的 [[Mega Cap]] / GB200 PSU。
3. [[X7R]] 是大宗消費品，毛利率拉不到 6173 自承的 28.4% / 目標 30%+。

**Verification log：**
- 三組獨立來源（TechNews 標題、biggo 法說摘要、TW 公告 5/15 capex 案）皆指向 [[NP0]] / [[C0G]]。
- 法說粒徑口徑 200nm；產業媒體 100nm 是延伸推演而非公司明文承諾。**寫入「100nm [[NP0]]」需註記「業界口徑 / 法說階段目標 200nm」。**

## Q2: 6173 是否供 Big-4（[[Murata]] / [[Taiyo Yuden]] / [[Samsung Electro-Mechanics]] / [[Yageo]]）008004 line？

### 結論：**沒有任何公開證據**指向粉體外銷 Big-4 008004。法說會與 IR 反向表態「**避開 0201**」。

### 證據

**法說會口徑（直接否定 008004 路線）：**
> 「信昌電的定位非常清晰，專攻 **1206 以上**規格的大尺寸產品，**避開日系廠商擅長的 0201 等微型化手機應用市場**」<br>
> 來源：https://finance.technews.tw/2026/05/15/high-end-np0

008004（0.25×0.125mm）比 0201（0.6×0.3mm）更微型，6173 既然避開 0201，自然不會去攻 008004。

**集團內部 captive 為主：**
- 6173 由 [[華新科]] 集團持股 43.13%
- 法說會：「華新事業群旗下被動元件廠 [[華新科]]、[[信昌電]]、[[瀚宇博德]]、[[精成科技]]、[[華東科技]]」垂直整合
- 來源：TEJ 財金研究院 https://medium.com/tej財金研究院/被動元件雙雄的頂上對決
- 推論：粉體一線優先供 [[華新科]] / 6173 自家 MLCC line，外銷量不大

**「日系客戶」可能身分推測：**
- 法說會未具名「日系客戶」
- 6173 主攻 1206 以上、車用、低軌衛星、人形機器人、HVDC
- 候選：[[Denso]]（車用 PSU 模組廠）、[[Aisin]] 工業電子、[[KOA]]（電阻廠也做 powder business？）
- 注意：**[[Murata]] 子公司不會買 6173 粉**——Murata 對 [[Sakai 化學]] 是長期 lock-in 關係，且 Murata 自己也做 in-house barium titanate
- **最可能候選：日系車用 Tier-1（[[Denso]]、[[Aisin]]、[[村田 PSU]]）採購 [[NP0]] 1206+ 規格大尺寸成品 MLCC**——非粉體外銷

### Hidden evidence 檢視

- **專利**：未在公開 USPTO / TIPO 找到 6173 與 [[Murata]] 共同申請或 cross-license 證據（本輪未深挖到專利資料庫，flag for 進階驗證）
- **MOPS 重大訊息**：法說會未揭露日系客戶名稱（受 NDA）
- **產業媒體**：「法人預期日系客戶轉單持續發酵」是 narrative，不是 hard evidence

**Verification log：**
- 008004 line 供應假設：**強烈不成立**（公司明文避開 0201）
- 大尺寸 1206+ [[NP0]] 供應 + 日系車用 Tier-1：**high probability**（未指名）
- 粉體外銷 Big-4：**low probability**（集團內部 captive 優先）

## Q3: 4760 銅漿 / 銀漿 D50 / D90 粒徑等級？

### 結論：**公開資料缺失**。4760 法說會與公司簡報**完全未揭露 D50 / D90 數據**。

### 證據

**唯一可用線索：競爭優勢敘述「配方客製化」而非粒徑**
- 「勤凱由於能配合客戶開發客製化電漿料」
- 「在客製化電漿料配方，過去這是日本業者最為擅長」
- 來源：https://www.bnext.com.tw/article/60871/ample-conductive-paste-2021

**這暗示什麼？**
- 4760 的競爭力**並非建立在 sub-100nm 鎳粉 / 銅粉自製粒徑**
- 而是建立在「**外採金屬粉 → 配方 / 分散劑 / 黏結劑客製化**」的調漿能力
- 這與 [[Shoei Chemical]] / [[Sakai 化學]] 的技術路徑**根本不同**

### 為何重要

- [[Murata]] 008004 line 對銅 / 鎳粉 D50 要求是 **80-150nm**（推估），D90/D50 比 < 2.0（窄分布）
- 4760 不揭露 D50/D90 意味著：
  - (a) 規格未達 008004 in-spec
  - (b) 規格達標但屬商業機密
  - (c) 4760 的真正 moat 是配方而非粒徑——所以粒徑不是重點
- **(c) 的可能性最高**——但這也意味著 4760 在「008004 in-spec 純度競爭」中**結構性弱勢**，因為粒徑是日系護城河

**Verification log：**
- D50/D90 完全 unknown
- 寫入報告請使用「規格未公開」「公司強調配方而非粒徑」描述，**禁用「sub-100nm」「達日系水準」等量化主張**

### 為何業界暗示 [[Murata]]

### 規格推測（無證據）

- 4760 在 [[國巨]] 銀膏 50%、[[奇力新]] 銀膏 60-70% 滲透——這些客戶以 0402 / 0603 大宗為主
- 切日系若以**最小阻力路徑**推，應從 **0603 / 0402 工業 / 車用** start，**非 008004 旗艦線**
- 008004 是 [[Murata]] 最高純度 line，絕不會輕易讓初認證的台廠切入

**Verification log：**
- 「日系認證」=「送件 + 初步認證」（非 in-spec 量產）
- 「日系龍頭」≈ [[Murata]]（口語推估，非公司確認）
- 規格 = 完全未公開，**absolutely 禁用 008004 字眼描述 4760 日系滲透**

## Q5: Ni paste TW 替代品檢視

### 結論：**TW 完全無自研 / 上市鎳粉鎳膏玩家**。[[科揚企業]] 是日本 [[MMS]]（Mitsui Mining & Smelting）的代理商而非製造商。

### 證據

**TPCIA 產業資料：**
> 「[[科揚企業]] 是被動元件領域材料代理銷售，代理日本 **[[MMS]]**（[[Mitsui Mining & Smelting]]）的 nano nickel slurry」<br>
> 來源：https://www.tpcia.org.tw/產品資訊/通路-代理-其他/科揚企業

**其他 TW 鎳粉相關公司（皆非 MLCC 內電極等級）：**
- [[長叡國際興業]] — 粉末冶金用 (μm 等級 ≫ nano)
- [[鴻鈞 / 宏泰應用材料]] — 銅粉為主
- [[富士多]] — 粗鎳粉 / 工業用途
- [[舶鈞]] — 奈米銅粉（非鎳）

### 結構性卡點

| 項目 | TW 現狀 | 日系現狀 |
|---|---|---|
| Nano Ni 粉 (sub-100nm) 自製 | **零上市玩家** | [[Shoei]]、[[Daiken]]、[[JFE Mineral]]、[[MMS]] |
| Ni paste / slurry | **零** | [[Shoei]]（與 [[Murata]] toll refining） |
| 替代 / 切入路徑 | 唯一代理 [[MMS]]（[[科揚]]） | n/a |

**這對 008004 thesis 的關鍵意涵：**
1. 鈦酸鋇粉部分，6173 至少在「自製 + 路線清晰」這格上有 marker
2. **但 BME 製程內電極（佔 MLCC 成本 35-50%）的 Ni paste，TW 沒有任何切入點**
3. 這意味著「TW 上下整合切 008004」的敘事**結構性受限於日本 Ni paste 寡占**
4. **6173 + 4760 + [[華新科]] / [[國巨]] 即使 100% 整合，也仍須向 [[Shoei]] / [[MMS]] 採購 Ni**——日系供應鏈卡點不可繞過

**Verification log：**
- 鎳粉自製：TW 0 玩家 → 結論高信度
- 鎳膏：TW 0 玩家 → 結論高信度
- **這是 prior batch 沒有充分強調的結構性風險，本輪 escalate 為 critical finding**
