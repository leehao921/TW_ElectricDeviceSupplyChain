# Unit 3: 台灣利基 — 3026 禾伸堂 + 6173 信昌電 的 008004 切入點

## TL;DR

- **[[3026 禾伸堂]] ([[Holy Stone]]) 與 008004 完全相反方向**：產品線聚焦 100V–10,000V 高壓、中壓 MLCC 與大尺寸高容量規格,目標市場是 [[NVIDIA]] Vera Rubin AI 電源、HVDC 電源供應器與車用電子;**官方產品目錄無 008004 (0.25×0.125mm) 規格**,技術 roadmap 也未公開朝該方向投資。2026Q1 EPS 創 4 年新高、交期 20+ 週的需求全部來自高壓而非微小化。對 008004 的直接 exposure **= 0**。
- **[[6173 信昌電]] (華新科集團子公司) 是「自製介電瓷粉的台灣唯一者」**,技術目標 200nm 微細化粉末 — 仍然比 008004 所需的 50–100nm 級 [[BaTiO₃]] 粒徑粗 2–4 倍。元件端主力是 1206 以上大尺寸、高壓 MLCC 與 Mega Cap 堆疊電容,與 008004 微小化路線**反向**。**B2B 供應 [[Murata]] / [[Taiyo Yuden]] / [[Samsung Electro-Mechanics]] 008004 產線的證據未出現於任何法說會或公開資料**;母公司 [[華新科]] 才是亞太區進階 MLCC 主力,信昌電的角色是材料 + 利基元件雙軌。
- 投資角度排序:**3026 對 008004 exposure = 0**;**6173 透過粉體技術潛在 exposure 有限 (粒徑差距 2–4 倍)**;兩家受惠的是「高壓 / 大尺寸 / Mega Cap」這個與 008004 完全不同的 AI 電源 supercycle 子題材。

## 3026 禾伸堂 — 高壓利基 vs 微小化反向

### 公司結構與市值

- 板塊: Electronic Components (Technology)
- 市值 22,146 百萬台幣 (約 NT$221 億);企業價值 20,457 百萬台幣 (來源: `Pilot_Reports/Electronic Components/3026_禾伸堂.md`)
- 2025 全年營收 13,427 百萬台幣 (YoY +5.0%);毛利率 18.77% (+243bp YoY);淨利 1,092 百萬台幣;淨利率 8.13%
- 雙軌獲利模式:**自有 [[MLCC]] 製造 (高壓 / 高容 / 特殊規格)** + **半導體與被動元件代理** ([[ROHM]]、[[Vishay]]、[[Azoteq]] 等)
- 股本與毛利率水準遠低於 [[國巨]] / [[華新科]],靠利基規格避開標準品紅海

### 產品線:**100V – 10,000V 高壓 + 大容值,不做 008004**

[[Holy Stone]] 自家功能陶瓷材料實驗室自主開發**高壓 (100V–10,000V)、中壓 (50V–100V) 及低溫飄係數**等特殊材料的 [[MLCC]]。其產品 catalog 明確以「高壓、高容、特殊規格」為定位,**避開消費電子標準品所需的微小化路線**。

008004 的物理規格 (0.25mm × 0.125mm、介電層 < 1μm、層數 > 600、典型耐壓 4–6.3V) 屬於**極端微小化 + 低壓**製程,需要的關鍵能力:
- **50–100nm 級 nano [[BaTiO₃]]** (3026 自有粉體粒徑公開資料指向標準型,非奈米級)
- **超薄介電層流延 (< 1μm)** + 高層數疊片 (> 600 層) (3026 量產經驗集中在中等層數高壓堆疊)
- **次微米級電極印刷與共燒**

**3026 在這三項能力上都無公開投資**;其資本支出 2023–2025 累計約 1,364 百萬台幣 (CAPEX 合計),全部投入高壓 [[MLCC]] 與 [[碳化矽]] 基板擴產。董事長 2026 股東會明確表示產能 2026/2027 各擴 30%,目標瞄準 [[NVIDIA]] Vera Rubin 世代的 **AI 電源 (高壓 [[MLCC]])**,並非微小化路線。([[豐雲學堂]] / [[經濟日報]] 報導, 2026/05)

### SiC AMB 副業 — 與 008004 無關

3026 近年發展 [[碳化矽]] 基板金屬化與 [[AMB 基板]] (Active Metal Brazing, 主動金屬接合基板),用於 [[電動車]] 與功率模組,與微型化 [[MLCC]] 屬於完全不同的供應鏈。此業務拉高毛利率 (2025 全年 18.77% 創 4 年新高) 但與 008004 主題無交集。

### 是否有任何小區段切入 008004?

**結論:無公開證據顯示 3026 量產或開發 008004**。理由:
1. **公司策略明示**:避開標準品紅海、聚焦高壓利基 → 與微小化方向衝突
2. **技術能力缺口**:無 nano [[BaTiO₃]] 公開研發紀錄,無 < 1μm 介電層流延設備投資披露
3. **客戶結構錯位**:其客戶 [[台達電]]、[[光寶科]]、[[中磊]]、[[智邦]] 與車用 Tier 1 都需要的是大尺寸高壓 [[MLCC]],非智慧型手機 / 穿戴式裝置 / [[AI 加速器]] PCB 嵌入用的 008004
4. **產能規劃明示**:2026/2027 擴產全部用於 AI 電源高壓 [[MLCC]],無微小化新線投資

**3026 對 008004 直接 exposure = 0**。

## 6173 信昌電 — 自製粉體 → 008004 材料端能力?

### 公司結構驗證 (CLAUDE.md Golden Rule #2)

- Ticker: 6173
- 公司全名: **信昌電子陶瓷股份有限公司** (Sinoceramics Inc.)
- 隸屬:**[[華新科]]技集團子公司** (母公司 2492 為亞太主要 [[MLCC]] 廠之一)
- 板塊:Semiconductor Equipment & Materials (Technology)
- 市值 10,529 百萬台幣 (約 NT$105 億);企業價值 9,458 百萬台幣
- 2025 全年營收 4,055 百萬台幣 (YoY +8.9%);毛利率 24.27% (+284bp YoY);淨利率 13.75%
- 文件路徑:`Pilot_Reports/Semiconductor Equipment & Materials/6173_信昌電.md` ✓

### 雙軌業務:介電瓷粉 + 利基元件

信昌電是**台灣唯一具備介電陶瓷粉末 ([[MLCC]] / LTCC / 微波瓷粉) 自製能力的廠商**。其垂直整合鏈為:

```
原料 (氧化鈦、碳酸鋇、稀土) → 自製 [[BaTiO₃]] 介電瓷粉 → 高壓 / 大尺寸 MLCC + 晶片電阻 + 電感
                                              ↓
                                       母公司 [[華新科]] 也採購其瓷粉
```

### 自製粉體的奈米化能力 — 200nm vs 008004 所需 50–100nm

關鍵技術 gap 分析:

| 項目 | 008004 量產所需 | 6173 信昌電現況 (公開資料) | 差距 |
|---|---|---|---|
| [[BaTiO₃]] 粒徑 | 50–100nm | **200nm 目標 (技術 roadmap)** | 2–4 倍粗 |
| 粒徑均勻度 | 極高 (< ±15%) | 未公開 | 未知 |
| 介電層厚度 | < 1μm | 利基大尺寸取向,公開資料未提及 < 1μm | 明顯 gap |
| 層數 | > 600 | 大尺寸高壓取向,層數中等 | 明顯 gap |

公開研究顯示**鈦酸鋇奈米粉體用絡合法可製出 < 70nm 粒徑** (來源: 國立交通大學論文 *含二氧化鈰之奈米鈦酸鋇之製備、微觀結構與介電性質之研究*),技術門檻可達,但 6173 法說會 (2026/05/21) 公開的技術目標是「**200nm 微細化粉末**」,**沒有公開 ≤ 100nm 級量產粉體的證據**。

200nm 級已足夠支撐:
- 0201 (0.6×0.3mm) 中高容值 MLCC
- 大尺寸 (1206 以上) 高壓 MLCC
- Mega Cap 堆疊電容

但**不足以直接量產 008004 介電膜** — 介電層厚度需 < 1μm,粒徑必須 ≤ 100nm 才能堆疊到 > 600 層而不產生缺陷。

### 元件端產品線:大尺寸高壓 — 與 008004 反向

信昌電 2026/05/21 法說會明確揭示策略 ([[優分析]]、[[鉅亨網]] 報導):

- **AI 電源從 12V → 48V → 800V HVDC 架構轉換**,需要**大尺寸 (1206 以上) + 高壓 [[MLCC]]**
- [[NVIDIA]] GB200 高階 AI 伺服器單機櫃 PSU 需求 1,400–1,800 顆,**Mega Cap 堆疊電容單價是標準品的 7–9 倍**
- 高階特殊品交期延長至 16 週、產能利用率 85–90%
- 多家 Tier 1 客戶洽談長約
- 砸 11.6 億台幣擴產搶 AI 與 HVDC 商機 (2026/05 公告, [[壹蘋新聞網]])
- 未來資本支出規劃 NT$15–20 億

**這條 roadmap 與 008004 微小化路線完全反向** — 信昌電的 AI 伺服器商機在「大」而非「小」。

### 與母公司 [[華新科]] 的技術分工

- **[[華新科]] (2492)**:亞太前段 [[MLCC]] 量產主力 (含 0402 / 0201 等中小尺寸標準品 + 高階品),自製粉體部分依賴信昌電 + 部分外購
- **[[信昌電]] (6173)**:介電瓷粉自製主力 + 高壓 / 大尺寸利基 + 晶片電阻 / 電感
- 集團內分工:信昌電供瓷粉、做利基規格;華新科做主流標準品 + 中高階規格量產
- **若未來集團要切入 008004,主導者會是華新科,而非信昌電** — 因為主流微小化規格量產設備、製程 know-how、客戶通路都在 2492

## B2B 供應 Big-4 證據檢視

### 信昌電 vs [[Murata]] / [[Taiyo Yuden]] / [[Samsung Electro-Mechanics]] / [[TDK]]

**檢視結論:無公開證據顯示信昌電供料給日韓 Big-4 的 008004 產線**。理由:

1. **Big-4 的 [[BaTiO₃]] 粉體幾乎全部內製**:
   - [[Murata]] 自有粉體 + 與 [[Sakai 化學]] 合資
   - [[Taiyo Yuden]] 自製粉體 + 投資 50 億日圓擴新廠
   - [[Samsung Electro-Mechanics]] 自製粉體 + 與 [[Nippon Chemical Industrial]] 合作
   - **無外購台灣瓷粉的公開紀錄** (尤其在最關鍵的 008004 / 01005 規格)

2. **信昌電法說會明示主要客戶**:
   - 母公司 [[華新科]]技集團 (採用其瓷粉)
   - 日系客戶 (未具名,推測為日系車用 / 工業客戶)
   - 全球 [[AI 伺服器]]、[[網通]]、車用 / 工業電子系統廠
   - **未提及供應 Big-4 的微小化產線**

3. **微小化粉體規格門檻**:Big-4 在 008004 量產用的 [[BaTiO₃]] 必須 ≤ 80nm 且粒徑分布極窄,屬於高端 know-how 護城河,**外購機率極低**,且台廠目前無公開資料達到此 spec。

4. **產業競合關係**:信昌電與華新科本身就是 Big-4 在中高階 [[MLCC]] 的競爭對手,Big-4 不會把關鍵粉體外包給競爭對手集團。

**信昌電對 008004 的潛在 B2B exposure = 接近 0**;若有,也只可能是極少量的試料或特殊規格 (例如非微小化的車用 MLCC 粉體),不構成 investment thesis 級的曝險。

## 投資角度:兩家對 008004 的 exposure 強度排序

| 公司 | 直接量產 008004 | 材料端 (粉體) 供 008004 | AI 電源主題 exposure (對比) |
|---|---|---|---|
| [[3026 禾伸堂]] | **0** | 0 | **強** (高壓 [[MLCC]] for [[NVIDIA]] Vera Rubin) |
| [[6173 信昌電]] | **0** | **極低** (粒徑 gap 2–4 倍) | **強** (大尺寸 + Mega Cap for [[NVIDIA]] GB200) |

### Mental model:不要把「AI 伺服器熱」與「008004 微小化熱」混為一談

- **008004 主題**:微小化 → 用於手機、穿戴、AI ASIC PCB 嵌入、HBM 載板被動元件 → 受惠者主要是日韓 Big-4 + 國巨高階線
- **AI 伺服器電源主題 (3026 + 6173 的甜蜜點)**:大尺寸高壓 → 用於 PSU、HVDC 架構、Mega Cap → 與 008004 物理上反向
- 兩個題材**規格反向、客戶不同、製程不同**,但都搭著「AI 伺服器」的傘標籤,容易在投資人敘事中被混淆

### 若用戶想要 008004 直接 exposure,3026 / 6173 都不是答案。

正確答案應該在:
- **[[國巨]] (2327)** — 高階線可能跟進 008004 / 01005 規格
- **日韓 Big-4 之台廠供應鏈** (設備、原料、檢測) — 例如 [[Sakai 化學]] 在台子公司、[[群創]] 載板嵌入 client、台廠精密設備出口商
- **[[華新科]] (2492)** — 集團中真正會做主流微小化量產的單位 (若集團要切入)

## Open questions

1. **信昌電是否有未公開的奈米粉體 R&D pipeline (≤ 100nm)?** 法說會只揭示 200nm 目標,但實驗室可能更前沿 — 需追蹤專利與學界合作 (例如 [[國立交通大學]] / [[國立陽明交通大學]] 鈦酸鋇研究團隊)。
2. **華新科集團整體會不會在 2027–2028 切入 008004?** 若切入,信昌電的角色從「200nm 粉體」升級為「100nm 粉體」可能性如何?
3. **3026 禾伸堂的 [[碳化矽]] / [[AMB 基板]] 副業會不會稀釋本業 MLCC 投資?** 需追蹤 2026/2027 CAPEX 分配。
4. **Big-4 是否曾外購任何台灣瓷粉?** 若有,屬於哪個 spec band? 需查 [[Murata]] / [[Taiyo Yuden]] 的供應商名單揭露。
5. **6173 法說會「日系客戶」是誰?** 若為日系 Tier 1 車用 (例如 [[Denso]] / [[Aisin]]),與 008004 主題無關;若為日系 [[MLCC]] 中游 (例如 [[KOA]] / [[Murata 子公司]]),則可能間接相關。
