# Unit 28: 半導體晶圓 + 記憶體 4 強 — 法人為何爆買

## TL;DR

- 本 unit 4 檔 (6770、5347、2344、2408) 在 5/16–5/29 區間，外資合計買超 **+636,396 張**、投信合計 **+44,734 張**，是 [[Vera Rubin BOM]] 之外的第二大「法人共振」主題；驅動因素是 **2026 [[DRAM]] / 利基記憶體景氣翻揚** + **8 吋特殊製程漲價週期**。
- **2344 [[華邦電]]** 是 unit 內「雙引擎最強」訊號 — 外資 +100,442 / 投信 +53,557 / 合計 +153,323 張，對應 Q4 2025 毛利率自 22.66% 跳升至 **41.86%** ([[NOR Flash]] + 利基 [[DRAM]] 同步價漲)，這是過去 8 季首次出現雙位數營益率。
- **6770 [[力積電]]** 外資 +364,135 張位居 unit 之冠 — 雖然全年仍虧損，但 Q4 2025 毛利率重回 **+6.36%** (前 3 季皆為負)，[[Micron]] 銅鑼 [[HBM]] 代工合作 + [[Tata Electronics]] 印度合資是中長期「折舊大降 + 重啟產能」邏輯。
- **2408 [[南亞科]]** Q4 2025 營收 **30,094 百萬台幣** (年化 1,200 億+，較 Q3 跳升近 3 倍)、毛利率 **49.05%**、單季淨利 **11,093 百萬台幣** — 標準型 + 利基型 [[DRAM]] 價漲共振，但外資 +72,607 / 投信 **-23,785**，投信仍對 [[HBM]] 進度持保留。
- **5347 [[世界先進]]** 外資 +99,212 / 投信 +12,742 — Buy thesis 是「8 吋 [[BCD]] / 高壓特殊製程漲價」+ [[VSMC]] 新加坡 12 吋廠 ([[NXP]] 合資) 2027 投產，受惠 [[電動車]] + [[AI 伺服器]] 電源管理需求。

## 6770 [[力積電]] — 外資 +364K 第一名為何?

### 法人 5/16–5/29 累計
| 項目 | 張數 |
|---|---:|
| 外資 | +364,135 |
| 投信 | +2,220 |
| **合計** | **+390,298** (unit 排名 #1) |

(註: 外資 + 投信 + 自營 ≠ 累計合計，係因自營商與避險買賣抵銷；以總額 +390K 為主訊號)

### 基本面 verify (Pilot_Reports/Semiconductors/6770_力積電.md)
- FY2025 營收 **46,730 百萬台幣** (YoY +4.5%)、毛利率 -3.29% (FY2024 +1.17%)、淨利 -7,813 百萬台幣 (連續第三年虧損)。
- **Q4 2025 毛利率重返 +6.36%** — 前 3 季分別為 -4.82%、-9.06%、-6.53%，Q4 為 2025 唯一正毛利季，是「景氣谷底翻揚」最直接訊號。
- Op CF FY2025 **+7,823 百萬台幣** (FY2024 +6,495)，CapEx 自 -23,533 大降至 -8,593 → **折舊壓力高峰已過**。

### Catalysts (法人 buy thesis)
1. **[[Micron]] 銅鑼廠 [[HBM]] 代工合作** — 2024 年公告 18 億美元出售銅鑼 12 吋廠予 [[Micron]]，雙方展開 [[HBM]] 後段封裝/前段代工合作，2026 進入認列期。([[HBM]] 是 [[Vera Rubin BOM]] 核心瓶頸。)
2. **[[Tata Electronics]] 印度合資 12 吋廠** — 技術授權 + 顧問費收入，[[AI]] 邊緣晶片 / [[ADAS]] 用 28nm-40nm 製程。
3. **[[DRAM]] cycle 翻揚 + 利基記憶體價漲** — 力積電擁有自有 [[DRAM]] 產能，可受惠 2026 [[DDR4]] / 利基 [[DRAM]] 價漲。
4. **折舊大降 + CapEx 縮減** — 2023–2024 兩年 [[CapEx]] 合計 -71,549 百萬台幣，2025 縮減至 -8,593，**2026 起折舊曲線下行** → operating leverage 強。
5. **[[LCD]] 驅動 IC + 邏輯代工結構優化** — 從 LCD 驅動 IC (約 40%) 往 [[AI]] / 車用 / 工業擴張。

### 受惠強度
**★★★★☆ (4/5)** — 法人最大買盤但全年仍虧損；屬「景氣谷底翻揚」深度價值買法，需 Q1–Q2 2026 毛利率持續轉正才能再上一階。

## 5347 [[世界先進]]

### 法人 5/16–5/29 累計
| 項目 | 張數 |
|---|---:|

| 投信 | +12,742 |

### 基本面 verify (Pilot_Reports/Semiconductors/5347_世界.md)
- FY2025 營收 **48,591 百萬台幣** (YoY +10.3%)、毛利率 **28.10%** (FY2024 27.09%)、淨利 **7,908 百萬台幣** (+12.2%)。
- 4 季毛利率 27.51% / 28.04% / 30.10% / 28.68% — 穩定在 27%-30% 區間，符合「8 吋特殊製程定價權」邏輯。
- FY2025 投資現金流 **-59,711 百萬台幣** ([[VSMC]] 新加坡 12 吋廠 CapEx 高峰)，FY2025 CapEx **-63,978 百萬台幣** — 反向訊號是「法人押注 2027 投產後的營收倍增」。

### Catalysts (法人 buy thesis)
1. **8 吋晶圓特殊製程漲價週期** — [[BCD]] / [[SOI]] / 高壓 / [[MEMS]] 在 [[AI 伺服器]] 電源管理 + [[電動車]] [[BMS]] 需求帶動下，2026 報價持續上修。
2. **[[VSMC]] 新加坡 12 吋廠 (與 [[NXP]] 合資)** — 2027 投產，鎖定車用功率元件 / [[BCD]]，2024 年公告為近年 unit 內最大資本支出案。
3. **[[台積電]] 持股 28% 護城河** — 製程技術轉移 + 客戶 referral，營運穩定性高於其他 8 吋同業。
4. **同業 [[力積電]] / [[聯華電子]] / [[Tower Semiconductor]] 結構性供給有限** — 全球 8 吋特殊製程擴產極少，供給端僵化。
5. **股息防禦力** — 殖利率 2.68%，相對 unit 內其他 3 檔 (0.32%-0.43%) 為純成長股，世界先進偏「穩定成長 + 配息」型。

### 受惠強度
**★★★★☆ (4/5)** — 體質最穩、毛利率最高，但 CapEx 大幅膨脹是中期 EPS 稀釋風險；法人買盤反映「2027 [[VSMC]] 投產 + 8 吋特殊製程漲價」雙引擎。

## 2344 [[華邦電]] (unit 雙引擎最強信號)

### 法人 5/16–5/29 累計
| 項目 | 張數 |
|---|---:|
| 外資 | **+100,442** |
| 投信 | **+53,557** |
| **合計** | **+153,323** |

**外資 + 投信同向最大** — 是 unit 內最罕見的「雙引擎」格式 (其餘 3 檔投信均小於 13K 或負值)。

### 基本面 verify (Pilot_Reports/Semiconductors/2344_華邦電.md)
- FY2025 營收 **89,406 百萬台幣** (YoY +9.6%)、毛利率 **34.89%** (FY2024 29.41%)、淨利 **3,962 百萬台幣** (FY2024 僅 601 → 大幅 turnaround)。
- **Q4 2025 毛利率跳升至 41.86%** (前 3 季 25.62% / 22.66% / N.A.)、營益率 **+15.57%** (前 3 季全為負) — 這是過去 8 季首次出現雙位數營益率，**單季變化幅度 unit 之冠**。
- Q4 2025 單季營收 26,625 百萬台幣 (Q3 → 27.0% 跳升)，反映 [[NOR Flash]] 與利基 [[DRAM]] 同步調漲。

### Catalysts (法人 buy thesis)
1. **[[NOR Flash]] [[AI 伺服器]] bridge memory 需求爆發** — [[Vera Rubin]] / [[Blackwell]] 平台 BMC / BIOS Flash 容量上修，華邦電全球 [[Serial NOR Flash]] 第一大廠直接受惠；[[NOR Flash]] 在 [[Vera Rubin BOM]] 已是隱形瓶頸。
2. **利基型 [[DRAM]] 價漲** — [[DDR3]] / [[DDR4]] 利基 [[DRAM]] 在 [[Samsung]] / Hynix / [[Micron]] 退出標準型轉攻 [[HBM]] 後，利基市場「供給結構性退縮」，華邦電 2026 報價持續上修。
3. **車用 / 工控 / [[ADAS]] 高可靠度應用占比提升** — 平均售價高於消費型，毛利率結構性改善的關鍵。
4. **CapEx 自高峰下行** — FY2024 CapEx -17,533 → FY2025 -6,870 (-61%)，折舊壓力 2026 起緩解。
5. **華新麗華集團資源 + [[環球晶]] 矽晶圓綁定供給** — 集團內 [[矽晶圓]] 供應安全度高。

### 受惠強度
**★★★★★ (5/5)** — unit 內**最強單一訊號**：法人雙引擎共振 + Q4 毛利率跳升 + [[Vera Rubin BOM]] 直接連動 + 利基 [[DRAM]] 供給退縮 → 4 條 thesis 全部 align。

### 基本面 verify (Pilot_Reports/Semiconductors/2408_南亞科.md)
- FY2025 營收 **66,587 百萬台幣** (YoY **+95.1%**, [[DRAM]] cycle turnaround 最劇烈一檔)、毛利率 **22.51%** (FY2024 -1.23%)、淨利 **6,614 百萬台幣** (FY2024 -5,083 → 全年扭虧)。
- **Q4 2025 單季營收 30,094 百萬台幣** (Q3 10,526 → 跳升 **+186%**)、毛利率 **49.05%**、單季淨利 **11,093 百萬台幣** — Q4 2025 單季 EPS 已超過 FY2025 全年 EPS，是 unit 內 cycle 反轉最劇烈標的。
- Op CF Q4 2025 **+15,505 百萬台幣** (Q3 -876)，現金流翻轉。

### Catalysts (法人 buy thesis)
1. **[[DRAM]] cycle 反轉確立** — 標準型 / 利基型同步漲價，南亞科是全球第四大 [[DRAM]] 廠、台灣最大 [[DRAM]] 製造商；[[Samsung]] / Hynix / [[Micron]] 將 capacity 移向 [[HBM]] 後，標準 [[DRAM]] 供給缺口讓南亞科填補。
2. **10nm 級 1A / 1B / 1C 製程自主** — 自 2024 年起不再依賴 [[Micron]] 技術授權，IP 風險解除 + 製程升級空間打開。
3. **[[台塑]] 集團資金後盾 + 台勝科 [[矽晶圓]] 集團內鎖定**。
4. **[[LPDDR5X]] / [[HBM]] 跨入時程** — 法人留意 [[LPDDR5X]] for [[AI 伺服器]] 之進展 (但時程尚未明朗)；**這也是投信轉賣 -23,785 張的原因** — 短期估值偏高，HBM 量產時點仍未確定。

### 受惠強度
**★★★★☆ (4/5)** — 基本面爆發力 unit 之冠，但法人不同調 (投信賣超) 顯示**估值已部分 price in** + [[HBM]] 進度尚有不確定性；定位為「核心 [[DRAM]] cycle 配置」而非 alpha 進攻倉。

## 警訊 / 排除

1. **[[南亞科]] 投信 -23,785 張** — 與外資對立，反映短期估值偏高 + [[HBM]] / [[LPDDR5X]] 量產時程尚未明朗。**法人共振未確立**，配置權重應低於 [[華邦電]]。
2. **[[力積電]] 全年仍虧損** — Q4 毛利率僅 +6.36%，若 2026 [[DRAM]] 價漲不持續，景氣谷底翻揚 thesis 將被推遲；屬高 beta 標的。
3. **[[世界先進]] CapEx 大幅膨脹** — FY2025 CapEx -63,978 百萬台幣 (FY2024 -15,953)，[[VSMC]] 投資高峰，中期 EPS 稀釋風險，2027 投產延遲是最大 downside。
4. **[[DRAM]] cycle 反向風險** — 若 [[Samsung]] / Hynix 在 2026 H2 加速擴產 [[HBM]] 之餘也回補標準 [[DRAM]] 產能，[[南亞科]] / [[力積電]] / [[華邦電]] 利基 [[DRAM]] 議價權將受壓。
5. **[[Vera Rubin]] BOM 排他性提醒** — unit 內 4 檔不在先前 batch 已涵蓋名單 (2330 [[台積電]] / 2317 [[鴻海]] / 2308 [[台達電]] / 2383 [[台光電]] / 3037 [[欣興]] / 2313 [[華通]] / 3533 [[嘉澤]] / 3324 [[雙鴻]]) — 本 unit 為「記憶體 + 8 吋特殊製程」獨立角度。
