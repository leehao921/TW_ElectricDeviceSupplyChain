# 3034 - [[聯詠]]

## 業務簡介
**板塊:** Technology
**產業:** Semiconductors
**市值:** 301,213 百萬台幣
**企業價值:** 254,688 百萬台幣

[[聯詠]]科技 (3034) 是全球領先的[[驅動 IC]] 設計公司，為[[聯華電子]]集團成員。主力產品為 [[LCD 驅動 IC]]（佔 2025 年營收 ~56%）及顯示器 [[SoC]]（~44%），涵蓋 [[AMOLED]] 驅動 IC、HII（高端影像智能）邊緣 AI 視覺 SoC、電視 SoC、安防 SoC。公司在大尺寸面板驅動 IC 市佔率全球領先，並積極投入兩條新成長路徑：(1) HII [[AI]] 視覺晶片（機器視覺、無人機、機器人），(2) 高效能運算 ASIC（TSMC N4P，[[ARM]] Neoverse CSS N2，搭配 [[CoWoS]] + [[HBM3E]]）。

**HII 視覺 SoC（AI 邊緣推論）：** 搭載自研 NPU（完全自有架構，非授權 VeriSilicon/CEVA），TOPS 數字從未對外公開。前代 NT98530 安防 SoC 採 CEVA SensPro2 SP500 DSP；HII 系列為下一代自研版本。2025 年法說會確認應用場景：無人機、機器人、算力盒、機器視覺，但無客戶名稱揭露。AI 整合產品佔 SoC 線營收約 20%。

**HPC ASIC 大賭注：** 與 [[ARM]] Neoverse CSS N2 合作，採 TSMC N4P、77.5×77.5mm 晶片、CoWoS + HBM3e、16 CPU 核心 + 4 加速器。2025/9 首批晶圓驗證成功（ARM 官方部落格確認）。投資規模：4nm 單案「逾百億台幣」；2nm 下一代專案另外 US$2 億。最快 2027 才有營收。

## 供應鏈位置
**上游（晶圓代工 / IP）：**
- **晶圓製造：** [[聯華電子]]（驅動 IC 主要）；[[台積電]] N4P（HPC ASIC）；[[台積電]] N3（未來路線圖）
- **IP 合作：** [[ARM]] Neoverse CSS N2（ASIC）；CEVA SensPro2（前代 SoC）

**中游（IC 設計）：**
- **驅動 IC + AI SoC：** **[[聯詠]]**（自研 NPU，HII 視覺 SoC；LCD/AMOLED 驅動 IC）

**下游（終端應用）：**
- **面板廠：** [[群創]]、[[友達]]、[[京東方]]、[[三星顯示]]、[[LG Display]]
- **品牌：** [[Apple]]（平板驅動 IC）、電視品牌
- **AI 邊緣應用（目標）：** 無人機、機器人、算力盒（法說會確認方向，無客戶名稱）
