# 7419 - 達勝

## 業務簡介
**板塊:** Technology
**產業:** Electronic Components
**市值:** 6,080 百萬台幣
**企業價值:** 4,790 百萬台幣

達勝科技 (7419，[[Mortech]]) 為台灣專業[[聚醯亞胺]] (Polyimide, PI) 薄膜製造商，擁有平均 10-30 年 PI 產業經驗的技術團隊，成立於 1993 年。

核心產品與應用：(1) 標準型 PI 膜 (PIF) — 應用於軟性印刷電路板 ([[FPCB]])、IC 封裝與馬達電器絕緣；(2) 改良型 PI 膜 (PIN) — 增強機械與熱性能之特殊 PI 膜；(3) 黑色 PI 膜 (PIB/PMB) — 應用於 [[FPCB]] 補強板與 COF 材料；(4) 散熱型 PI 膜 (PIT) — 可製成人工石墨散熱片，應用於智慧型手機與平板散熱。公司持續擴充 PI 新產能，積極拓展 FCCL 基板、COF 材料等 [[FPCB]] 主流應用市場。

## 供應鏈位置
**上游 (化工原料):**
- **PI 樹脂:** [[聚醯亞胺]]樹脂原料供應商
- **溶劑:** 特殊化學溶劑
- **設備:** PI 膜塗佈與延伸設備

**中游 (PI 膜製造):**
- **標準 PI 膜:** **達勝** ([[FPCB]]/IC 封裝用 PI 膜)
- **黑色 PI 膜:** **達勝** ([[FPCB]] 補強板/COF 用黑色 PI)
- **散熱 PI 膜:** **達勝** (人工石墨散熱片用 PI 膜)

**下游 ([[FPCB]] 與電子產業):**
- **FPCB/FCCL:** [[FPCB]] 與 FCCL 基板製造商 — PI 基材
- **IC 封裝:** IC 封裝廠 — 封裝用 PI 膜
- **散熱模組:** 散熱模組廠 — 人工石墨散熱片
- **馬達電器:** 馬達與電器廠 — 絕緣用 PI 膜
